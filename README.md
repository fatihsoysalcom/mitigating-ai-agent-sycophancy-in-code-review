# Mitigating AI Agent Sycophancy in Code Review

This example demonstrates the concept of AI sycophancy (yaltaklanma) in code reviews and how to mitigate it. It compares a standard, eager-to-please agent with an objective, anti-sycophantic agent using system prompt engineering, showing how to force AI agents to say 'no' to insecure code.

## Language

`python`

## How to Run

Run 'python anti_sycophancy_agent.py' to see the simulated comparison. To use real LLM responses, export your API key first: 'export OPENAI_API_KEY=your-key-here'.
